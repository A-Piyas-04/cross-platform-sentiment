# Data

This project expects two CSV files in this folder (not committed to the repo):

- `Reddit_Data.csv` — expects a `clean_comment` column (text) and a `category` column (label).
- `Twitter_Data.csv` — expects a `clean_text` column (text) and a `category` column (label).

Add the source/link you downloaded these from here, e.g. the original Kaggle dataset URL,
so anyone cloning the repo can reproduce your results.

Processed/balanced versions (`reddit_processed.csv`, `twitter_processed.csv`) are generated
by running the notebook and are also excluded from version control.
